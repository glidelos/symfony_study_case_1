# DOCKERIZATION 4 SYMFONY

Ultra Basic base for dockerization Symfony.

## Basic Requirement
- docker-cli
- docker-compose

## Installation

```
git clone https://github.com/glidelos/symfony_docker.
```


## Usage

```
#at first time:
docker-compose up -d --build
#next
docker-compose up -d
```
